Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis we pulled data from the OpenWeatherMap API and assembled a dataset on over 500 cities. The analysis involved exploring the effect the differing Latitudes had on Temperature, Wind, Cloudiness, and humidity.
